# Chonky gif assets

Drop your capybara gifs here using these exact filenames. The app falls back to a 🦫 placeholder for any file that's missing, so you can add them one at a time.

| Filename          | When it's shown                          |
| ------------------ | ----------------------------------------- |
| `idle-happy.gif`   | Default idle state, all meters healthy    |
| `idle-sad.gif`     | Idle state when any meter is low (<=20)   |
| `eat.gif`          | Playing for a couple seconds after Feed   |
| `sleep.gif`        | Playing for a couple seconds after Sleep  |
| `bath.gif`         | Playing for a couple seconds after Bath   |
| `play.gif`         | Playing for a couple seconds after Play   |
